查询订单信息列表(分页)

请求信息

请求地址

http://127.0.0.1:7086/api/v1/order/queryOrders

请求方法

POST

请求体类型

application/json

入参

入参示例 (RequestBody)

{
  "pageNumber": 1,
  "pageSize": 20,
  "orderNo": "2022050925939",
  "userId": 1,
  "orderType": 2,
  "orderStatus": 0,
  "orderTimeStart": "2020-05-09",
  "orderTimeEnd": "2020-05-09"
}

入参字段说明

字段 类型 必填 含义 其他参考信息
pageNumber
 Integer
 是
 分页-当前页码 示例值为 1
pageSize
 Integer
 是
 分页-分页大小 示例值为 20
orderNo

 否
 订单编号 示例值为 2022050925939
userId

 否
 用户ID 示例值为 1
orderType
 Integer
 否
 订单类型
1:购买会员订单
2:购买商品订单		 示例值为 2
orderStatus
 Integer
 否
 订单状态
0:待处理
1:已支付
2:已取消
3:已退款		 示例值为 0
orderTimeStart
 String
 否
 下单时间范围-开始
格式: 2022-05-09		 示例值为 2020-05-09
orderTimeEnd
 String
 否
 下单时间范围-结束
格式: 2022-05-09		 示例值为 2020-05-09
