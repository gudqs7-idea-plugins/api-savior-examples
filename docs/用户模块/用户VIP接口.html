查询VIP用户信息列表(分页)

请求信息

请求地址

http://172.24.144.1:7086/api/v1/vip/queryVipUsers

请求方法

POST

请求体类型

application/x-www-form-urlencoded

入参

入参示例 (Postman Bulk Edit)

pageNumber:1
pageSize:20
//searchKeyword:模糊搜索词23
//gender:2
//vipType:3
//firstOpenTimeStart:2022-05-09
//firstOpenTimeEnd:2022-05-09
//vipExpireTimeStart:2022-05-09
//vipExpireTimeEnd:2022-05-09

1: 男

vipType

 否

1:青铜
2:白银
3:黄金
firstOpenTimeStart
 String
 否
 VIP开通时间范围-开始
格式: 2022-05-09
firstOpenTimeEnd
 Date
 否
 VIP开通时间范围-结束
格式: 2022-05-09
vipExpireTimeStart
 String
 否
 VIP过期时间范围-开始
格式: 2022-05-09
vipExpireTimeEnd
 Date
 否
 VIP过期时间范围-结束
格式: 2022-05-09
